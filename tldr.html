Train your own State of the Art semantic web scraper!

Much of a data scientist's data may come from web scraping. We all know this process can be quite brittle when things change. This notebook shows how to use powerful OCR-free and semantic segmentation technologies to build a tool that can interact with websites automatically and robustly.
Serverless ML is a new powerful skill to master

Once a systems technology reserved for the largest tech organizations, serverless ML is now available for many of use through services like banana.dev. This list of projects shows different systems that can be implemented well on serverless technologies! If you're looking for a new way to learn, these might serve as inspiration.

Nvidia researchers develop text driven animation for game development

With all the chatter around generative text models, it's cool to see a use case in a very visual realm. Gaming assets are expensive to make and often require endless customizations. A team at NVIDIA developed a text driven animation system that combines reinforcement learning, language models, and a beautiful physics simulator to produce character animations on the fly.
The strongest open source LLM under the microscope

FlanT5 is widely considered to be the best open source language model. It was instruction-tuned which gives it a wealth of tasks it can successfully complete. However, that process was somewhat opaque. With The Flan Collection, we can now see the tasks, prompts, templates, and more that went into training the model we all have started using.
